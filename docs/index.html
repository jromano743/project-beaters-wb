El bot "uwu"

"Hace mucho tiempo, los usuarios y los bots de cierto servidor vivian en armonía. Pero todo cambió cuando los bots no tenian todas las funciones que el server queria.
Solo Juanma, 'maestro' de los lenguajes podia solucionarlo, pero cuando el mundo mas lo necesitaba. Se fue de viaje...
Despues de semanas Juanma logró traer a una solcion, un nuevo bot llamado Beaters. Aunque sus comandos eran increibles, tenia muchos bugs que solucionar antes de poder ser un autentico bot.
Y yo creo que Beaters podrá salvarnos... "

Lista de comandos

Beaters tiene muchos comandos y todos se pueden consultar desde el bot con el comando bt!help y de manera particular con el comando bt!help nombre_comando. Sin embargo aqui hay una lista de los comandos separados por categorias

Divertidos

Estos comandos te permiten hacer ciertas cosas cuando estas aburrido en la sala de chat.

*u => Etiquetar a un usuario
*m => Un mensaje
*n => Un numero
Todos los comandos se utilizan anteponiendo el prefijo del bot "bt!"

  • 8ball *m

    Hazle una pregunta a la bola magica.

  • alberto *m

    Deja que Alberto notifique cosas en el servidor.

  • happy

    Transmite tu felicidad!.

  • choose *m ; *m

    Si no puedes decidir algo, deja el bot lo haga por ti.

  • dice *n *n

    El bot tirará una cierta cantidad de dados. Cada dado tendrá la cantidad de caras que especifiques.

  • news *u *m

    Que hiciste el dia de hoy?.

  • love *u

    Utiliza la calculadora del amor con cierta persona <3.

  • say *m

    Deja que sea el bot el que diga lo que no quieres decir.

  • srole *u

    Etiqueta de 1 hasta 5 usuarios para que el bot les escoja una posicion de lol.

  • trap *u *m

    Utiliza tu carta trampa para derrotar a tu enemigo.
